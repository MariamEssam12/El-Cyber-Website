# Cyber-Website
An old game website with a GameTik feature that allows you to play web streamed games on the run.


Describing one of the games:

Connect Four is a classic two-player strategy game played on a vertical grid. The goal is to connect four of your colored discs in a row—horizontally, vertically, or diagonally—before your opponent does.

Game Setup:

The game is played on a 6-row by 7-column grid.

Each player chooses a color.

Players take turns dropping one of their colored discs into any of the seven columns.

The disc falls to the lowest available spot in that column

Technologies Used:

Heuristic algorithm is used for AI feature
